Gabarito do Quiz sobre Barragens

  1. 1. Qual é o propósito principal de uma barragem?

    Resposta correta: D) Todas as anteriores

  2. 2. Qual é o principal material usado na construção de barragens de concreto?

    Resposta correta: C) Concreto

  3. 3. Quais são os principais riscos associados às barragens?

    Respostas corretas: A) Ruptura, B) Inundação, C) Deslizamento, D) Todos os anteriores

  4. 4. O que é um plano de evacuação?

    Resposta correta: C) Um plano para evacuar pessoas em caso de emergência

  5. 5. Qual é o principal tipo de barragem usada para armazenamento de água potável?

    Resposta correta: A) Barragem de concreto

  6. 6. Quais são os principais componentes de uma barragem de terra?

    Resposta correta: B) Terra e rochas

  7. 7. Qual é o principal objetivo de uma barragem de rejeitos de mineração?

    Resposta correta: C) Armazenar resíduos

  8. 8. O que é um vertedouro de barragem?

    Resposta correta: A) Uma abertura para escoamento de água

  9. 9. Como são classificadas as barragens de acordo com sua função?

    Resposta correta: C) Barragens de contenção e barragens de armazenamento

  10. 10. Por que as barragens podem representar um risco para o meio ambiente?

    Respostas corretas: A) Alterações no ecossistema aquático, B) Deslocamento da fauna e flora, C) Liberação de poluentes, D) Todos os anteriores

  11. 11. O que é uma barragem de ensecadeira?

    Resposta correta: A) Uma barragem temporária

  12. 12. Quais são os principais métodos de controle de erosão em barragens?

    Respostas corretas: A) Cobertura vegetal, B) Gabiões, C) Terraplenagem, D) Todos os anteriores

  13. 13. O que é uma barragem de encaixe?

    Resposta correta: D) Uma barragem para contenção de resíduos

  14. 14. Qual é o principal método de construção de barragens de enrocamento?

    Resposta correta: C) Utilização de rochas e pedras

  15. 15. O que é uma barragem de enrocamento compactado?

    Resposta correta: C) Uma barragem de enrocamento com rochas compactadas
